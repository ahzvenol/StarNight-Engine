## 关于引擎 优势和特点
### 跨平台
基于浏览器运行，无需下载安装即可随时随地游玩，也可以方便的打包为应用程序分发到各个平台
### 简单易用
格式统一，易学习的中文剧本，并提供代码提示功能来提高编写效率，通过配置别名，你还可以将剧本中可能的重复部分转换为单个字符来进一步简化它
### 功能丰富
支持主流GalGame的几乎全部配置项和游戏效果
### 高自由度
- 基于配置文件，可以通过简单的编写满足大部分游戏逻辑外的需求，以后也将推出可视化的制作程序
- 允许自定义命令，如果你有编程基础，则可以通过编写JavaScript脚本来为你的游戏添加效果，并轻松的在剧本中使用它
- 如果这些都不能满足你的需求，还可以通过下载本项目源代码的方式来修改核心逻辑
### 基于移植
星夜引擎为移植而编写，开发者就是第一个用户，可以从用户的角度发现问题，解决问题，并在移植过程中不断优化引擎，增加功能
