## 关于引擎 优势和特点
### 跨平台
基于浏览器运行,无需下载安装即可随时随地游玩,也可以打包为应用程序分发到各个平台
### 简单易用
你可以使用任意你习惯的语言编写剧本
### 功能丰富
支持主流GalGame的配置项和游戏效果
### 高自由度
可以直接修改引擎的界面,扩展引擎的功能而无需重新编译代码
### 基于实践
星夜引擎基于对现有GalGame的分析和重制,在此过程中积累了大量经验
